# Guide des Patterns Rythmiques

Ce document explique tous les patterns disponibles dans Rhythm Trainer, classés par complexité.

## Système de Complexité

| Niveau | Type de subdivisions | Description |
|--------|---------------------|-------------|
| **1** | Noires/Croches basiques | Temps réguliers, faciles à suivre |
| **2** | Croches complètes + groupements | Toutes les croches, patterns simples |
| **3** | Syncopes + Subdivisions mixtes | Contretemps, mélange noires/croches/doubles croches |
| **4** | Doubles croches + Patterns complexes | Subdivisions rapides, syncopes avancées |
| **5** | Triolets + Aksak complexe | Patterns très asymétriques et rapides |

## Glossaire Musical

### Subdivisions
- **Noire** : 1 temps (en 4/4 ou 7/4)
- **Croche** : 1/2 temps (2 croches = 1 noire)
- **Double croche** : 1/4 temps (4 doubles croches = 1 noire)
- **Triolet** : Division du temps en 3 (au lieu de 2 ou 4)

### Concepts
- **Syncope** : Note qui tombe entre les temps (sur le contretemps)
- **Groupement** : Division d'une mesure en sous-groupes (ex: 2+3 divise 5 en 2 puis 3)
- **Aksak** : Rythmes balkaniques avec groupements asymétriques (2+2+3, 3+2+2, etc.)

### Exemples célèbres
- **5/4 (3+2)** : "Take Five" de Dave Brubeck
- **7/4 (4+3)** : "Money" de Pink Floyd
- **7/8 (2+2+3)** : Musique traditionnelle bulgare
- **9/8** : Couramment utilisé en jazz et musique progressive

## Progression Recommandée

1. Commencer par les patterns de **complexité 1** pour comprendre la signature rythmique
2. Passer aux **groupements simples** (complexité 2) pour sentir les accents
3. Explorer les **syncopes** (complexité 3) pour développer le sens du contretemps
4. Maîtriser les **doubles croches** (complexité 4) pour la rapidité
5. S'attaquer aux **triolets et patterns aksak** (complexité 5) pour l'expertise

### Conseils
- Commencez à **tempo lent** (40-60 BPM)
- Augmentez progressivement le tempo de **5 BPM**
- Visez une **précision ≤ 15ms** avant de passer au niveau suivant
- Utilisez le mode **Métronome + Pattern** au début pour entendre le rythme
- Passez au mode **Métronome seul** pour tester votre indépendance
- Essayez le mode **Silencieux** pour le challenge ultime
